|   | Arrays| Lists|
|---|---|---|
|Reading|O(1)|O(n)|
|Insertion|O(n)|O(1)|
|Deletion|O(n)|O(1)|

### Arrays
- random access
- elements are stored next to each other
- allows fast read
- all elements should be the same type

### Lists
- sequential access
- elements everywhere, each elements stores the address of the next element
- allows fast insertion and deletion
